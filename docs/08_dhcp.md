# 08 - Servicio DHCP

## Objetivo

Implementar un servidor DHCP para asignar automáticamente la configuración de red a los equipos del laboratorio.

---

## Configuración

- Rango de direcciones: 192.168.10.20 - 192.168.10.100
- Máscara: 255.255.255.0
- Puerta de enlace: 192.168.10.1
- Servidor DNS: 192.168.10.10
- Dominio: techsolutions.local

---

## Resultado

Los equipos cliente obtienen automáticamente una dirección IP válida y la configuración necesaria para integrarse correctamente en el dominio.
